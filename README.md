# API manager - Chas Academy recipe website project 🍽

A Vue.js-based website created for a <a href="https://github.com/Fabian144/ChasRecipeProject">recipe project<a/>, for each group to be able to manage the recipes in their REST API.

**Live demo:** https://fabian144.github.io/recipe-project-api-manager/

## Features 📦

- **Add recipes**: Add any amount of recipes at a time by pasting in an array in JSON format 
- **Delete recipes**: In the same way, delete any amount of recipes at a time
- **Add and change images**: Easily add or change the image for each recipe by pasting in an image URL or path

## Technologies used 🛠

- **HTML**
- **CSS**
- **JavaScript**
- **Vue.js**
- **Pinia** 
- **REST API**
